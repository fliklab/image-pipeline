# Mock Image Pipeline Dev Dashboard

## Overview
Daily mock cron: Mock news → summary JSON → PIL image → local save. Dev dashboard: progress, experiments, artifacts.

## Steps
1. Mock news data
2. Summary mock
3. Image mock (PIL)
4. Register
5. Pipeline
6. Scheduler (button)
7. Dashboard (Streamlit)

## Dashboard Layout
- Progress bar
- Pipeline tree
- Experiment table + gallery
- Logs
